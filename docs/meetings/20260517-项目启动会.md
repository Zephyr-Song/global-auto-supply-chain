# 5月17日 项目启动会议纪要

## 会议信息
- **日期**: 2026年5月17日
- **主题**: 汽车全球市场调研和跨国供应链 — 校外科研项目启动会

## 会议内容

### 1. 项目目标
基于大语言模型（LLM）构建全球汽车市场数据调研与跨国供应链风险分析系统。

### 2. 核心研究方向
- 全球汽车市场数据爬取与整合
- 供应链风险分析与预警
- 可视化交互展示
- 零碳电力投资系统数据库升级

### 3. 成员分工
各成员根据研究背景选择具体方向，详见启动会原始纪要。
