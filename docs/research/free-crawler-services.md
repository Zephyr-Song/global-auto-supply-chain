# 爬虫服务调研报告

## 免费爬虫服务对比

### 1. Apify
- **免费额度**: 每月 $5 免费额度
- **优势**: 内置 anti-scraping 绕过，Actor 市场丰富
- **劣势**: 免费额度有限，Akamai 级别站点可能需要付费 Actor
- **适用**: 样本数据采集、原型验证

### 2. Scrapy + Scrapoxy
- **成本**: 免费（需自建代理池）
- **优势**: 完全可控，Python 生态
- **劣势**: 需要维护代理IP池，对 Akamai 防御效果有限

### 3. Playwright/Puppeteer 半自动化
- **成本**: 免费
- **优势**: 模拟真实用户行为，可绕过部分 JS 检测
- **劣势**: 速度慢，需要人工介入

### 4. Octoparse
- **免费额度**: 10K 页/月
- **优势**: 可视化配置，无需编程
- **劣势**: 对高防站点支持有限

## 推荐方案
**分阶段策略**:
1. Phase 1: Apify 免费额度 → 样本数据
2. Phase 2: Playwright 半自动化 → 扩大数据量
3. Phase 3: 评估付费方案或替代数据源
